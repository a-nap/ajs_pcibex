Sie können die Bewertung durch Klicken mit der Maus oder durch einen Tastendruck mit den Tasten 1-7 abgeben. Sobald Sie den Satz bewertet haben, geht es automatisch weiter und Sie können die Bewertung nicht mehr ändern. Sie könnten zum Beispiel diesen Satz lesen:

Michael studiert Jura.

Dieser Satz wird in der Regel als natürlich empfunden. In so einem Fall würden Sie dem Satz eine hohe Bewertung geben. Sie könnten auch zum Beispiel diesen Satz lesen:

Juliane beschimpft die Schale.

Dieser Satz wird in der Regel als weniger natürlich empfunden. In so einem Fall würden Sie dem Satz eine vergleichsweise niedrige Bewertung geben. Es kann sein, dass ganz viele Sätze für Sie natürlich klingen. Es kann aber auch sein, dass nur die wenigsten für Sie natürlich klingen. Bitte versuchen Sie, so weit es geht, die gesamte Skala zu verwenden.

Zuerst werden Sie die Aufgabe anhand von 9 Beispielen üben, dann folgt das eigentliche Experiment. Danach werden Sie Ihren persönlichen Code für die Vergütung bekommen sowie eine kurze Erklärung zum Zweck dieser Studie lesen. Das gesamte Experiment dauert etwa eine Stunde. Die genaue Dauer hängt davon ab, wie schnell Sie lesen und ob Sie Pausen einlegen.
